# fenix-passwordmanager
A modern lb-phone password manager application for FiveM, built with a modern dark/light theme UI

<img width="971" height="662" alt="photo4" src="https://github.com/user-attachments/assets/35fb9edd-02c8-4cd2-b343-ddafb83516ad" />

# FEATURES
- Save and edit user
- Save and edit password
- See and copy user and password
- Autoensure sql
- Compatible with QBox

# Requirements
- lb-phone
- oxmysql

# Installation
- Download or clone this resource into your server's resources folder.
